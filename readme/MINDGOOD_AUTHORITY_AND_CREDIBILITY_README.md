# MindGood Authority, Reliability, and Credibility README

## Purpose

This document outlines the highest-impact actions MindGood can take to improve:

- user trust
- brand credibility
- clinical reliability
- search visibility for health-related queries
- AI/search retrieval confidence

MindGood operates in a health and mental wellness space, which makes this a high-trust and high-scrutiny category. For these topics, authority is not built mainly through SEO tactics alone. It is built by making the platform clearly trustworthy, clinically responsible, and easy to verify.

## Core Principle

MindGood should be unmistakably credible to both:

- real users deciding whether to book therapy
- search engines and AI systems deciding whether to trust and surface the brand

That means we need strong signals in four areas:

1. Identity trust
2. Clinical trust
3. Content trust
4. Technical trust

## Priority 1: Identity Trust

These items should be standardized first across the entire website, structured data, social profiles, and external listings.

### Requirements

- Use one official support email everywhere
- Use one official support phone number everywhere
- Use one legal entity name everywhere
- Use one official address everywhere
- Use one consistent brand description everywhere
- Ensure `mindgood.life` and `www.mindgood.life` are treated consistently in canonicals, sitemap, search console, and payment settings

### Why this matters

Conflicting business details make the platform look unreliable to users and to search systems. In a health-related space, inconsistent contact details reduce trust quickly.

### Recommended implementation

- Audit footer, contact page, structured data, booking pages, privacy/terms pages, Google Analytics labels, Open Graph metadata, and social profiles
- Replace all mismatched emails, phone numbers, and organization names
- Create a single source of truth in code for business identity fields

## Priority 2: Therapist Verification and Clinical Trust

Therapist profiles should be the strongest trust asset on the site.

### Every therapist profile should include

- full name
- professional title
- years of experience
- license number
- licensing authority
- license region or country
- languages spoken
- therapy modalities such as CBT, ACT, trauma-informed therapy, couples therapy
- areas of specialization
- who the therapist is best suited for
- verification status
- date verified by MindGood
- profile last updated date

### Recommended supporting pages

Create and link these pages sitewide:

- `How We Verify Therapists`
- `Clinical Standards`
- `Who MindGood Is For`
- `When MindGood Is Not The Right Fit`
- `Emergency and Crisis Support`

### Why this matters

Users need confidence that therapists are real, licensed, and reviewed carefully. Search engines and AI systems also rely on these signals to determine whether profile pages are authoritative.

## Priority 3: Content Credibility

MindGood’s concern pages and blog content should feel medically responsible and clearly sourced.

### Add to every concern page and blog post

- author name
- reviewer name where appropriate
- reviewer credentials
- published date
- last reviewed date
- why the content was created
- references or sources where medical or therapeutic claims are made

### Add these governance pages

- `Editorial Policy`
- `Clinical Review Policy`
- `Sources and Referencing Policy`

### Why this matters

Health-related content is held to a higher trust standard. Users and search engines should be able to answer:

- Who wrote this?
- Who reviewed this?
- Why should this be trusted?
- When was this last reviewed?

## Priority 4: Technical Trust

The site should expose clear structured and crawlable trust signals.

### Current focus areas

- Keep therapist pages server-rendered and crawlable
- Keep language and specialization directory pages crawlable
- Include relevant therapist, language, and specialization URLs in the sitemap
- Use consistent canonicals
- Improve page speed and mobile usability

### Structured data recommendations

Continue strengthening:

- `Organization`
- `MedicalBusiness`
- `ProfilePage`
- `Person`
- `MedicalWebPage`
- `BreadcrumbList`

Recommended fields to emphasize:

- `mainEntity`
- `reviewedBy`
- `knowsLanguage`
- `areaServed`
- `medicalSpecialty`
- `hasOccupation`
- `worksFor`
- `lastReviewed`

### Why this matters

Technical clarity helps search systems confidently understand:

- who MindGood is
- who each therapist is
- what languages and concerns each therapist supports
- which pages should rank for which user intents

## Priority 5: Clear Clinical Boundaries

MindGood should clearly explain what it can and cannot provide.

### Add visible disclaimers for

- non-emergency use
- crisis situations
- suicide/self-harm emergencies
- psychiatric emergencies
- situations requiring immediate local support

### Add clearer distinctions between

- psychologist
- therapist
- counsellor
- psychiatrist
- coach

### Why this matters

Clear boundaries increase credibility. A trustworthy mental health platform does not overclaim and does not blur emergency care with standard therapy support.

## Priority 6: Conversion Trust Signals

People deciding whether to book need reassurance before they act.

### Add trust blocks to homepage, therapist pages, and booking pages

- verified therapist badge explanation
- secure booking / secure payment explanation
- confidentiality explanation
- session format explanation
- multilingual support explanation
- what happens after booking
- cancellation/rescheduling clarity

### Optional but valuable additions

- testimonials
- anonymized reviews
- “featured in” mentions
- affiliations
- therapist count
- language count
- response time expectations

## Priority 7: External Authority Building

Authority is also built off-site.

### High-value external trust work

- professional therapist LinkedIn profiles
- therapist listings in credible directories
- university, wellness, HR, and expat community mentions
- expert quotes in local media
- podcast interviews, webinars, and workshops
- partnerships with organizations in UAE, GCC, expat, and mental health spaces

### Strong authority assets to create

- UAE expat mental health guides
- multilingual therapy access pages
- original insights on therapy demand by language or concern
- workplace mental health resources for GCC employers

### Why this matters

When third-party sites consistently reference MindGood and its therapists, search engines and AI systems gain more confidence in the brand.
